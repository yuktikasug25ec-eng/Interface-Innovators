Our Projects

Self-Balancing Bot (2024–2025)

A robot that maintains its balance dynamically using sensors and control algorithms.

Description: A two-wheeled robot that balances upright using feedback from gyroscope/accelerometer sensors. It demonstrates control theory fundamentals by dynamically adjusting wheel motion to keep its center of gravity stable using closed-loop PID control.
Year: 2024–25 (recent club build cycle)
Technologies: Arduino/ESP32, MPU-6050 IMU, DC motors with encoders, PID control algorithms, C/C++ programming, battery power.

MARK 1.6 Wireless Bot(2024)

An upgraded version of a modular robot platform (likely for motion/navigation tasks).

Description: A modular, Bluetooth-controlled mobile robot platform used for demos and testing autonomous functions. It is designed to be controlled via mobile or controller, providing a base platform for sensor integration and navigation experiments.
Year: 2024
Technologies: Bluetooth modules (HC-05/HC-06), Arduino/ESP32, motor drivers (L298/NFET H-bridge), mobile UI/BT pairing, chassis fabrication.

PLUTOX (Autonomous Bot) (2025)

A specialized robotics platform (name suggests an autonomous or mission-oriented robot).

Description: A mission-oriented robot built for navigation challenges and campus events. PLUTOX integrates basic line or obstacle detection to run autonomously or via controller, showcasing path-following and sensor fusion.
Year: 2025
Technologies: Line trackers, ultrasonic sensors, microcontrollers (Arduino/ESP32), DC/servo motors, logic control, PCB prototyping.

Automated Bot(2024)

General autonomous robotic system showcasing basic automation functions.

Description: A general autonomous robotic unit created to explore automation principles. It typically includes simple environment sensing and decision rules — for example, obstacle avoidance or task execution (like object retrieval).
Year: 2024
Technologies: IR/ultrasonic sensors, microcontrollers, motor drivers, basic path-planning logic, embedded C++.

Radar Navigation System(2025)

A project integrating radar sensors for navigation or obstacle detection.

Description: A project designed to integrate radar or ultrasonic ranging for robot navigation, enabling advanced obstacle detection for autonomous mobility beyond basic sensors — useful for indoor/outdoor robotics.
Year: 2025
Technologies: Radar/SONAR modules, microcontrollers (ESP32/RPi), signal processing algorithms, SLAM-style localization concepts, ROS-lite testing.

Hexapod Robot(2025)

A six-legged walking robot for advanced locomotion and terrain handling.

Description: A six-legged biomimetic robot capable of walking over varied surfaces. It allows experimentation with gait algorithms and advanced motion control, useful for terrain-aware robotics research and competitions.
Year: 2025
Technologies: Servo motors (18+), custom chassis fabrication/3D parts, gait kinematics, remote control/ROS nodes for motion, wireless communications.

ESP32-Based Autonomous Drone (IRoC 2025 Challenge)(2025)

A custom drone built using ESP32 and Raspberry Pi or Jetson Nano for robotics challenges, focusing on autonomous control and embedded intelligence (used in the IRoC 2025 Challenge).

Year: 2025
Description: Designed around an ESP32 + higher-compute board (e.g., Raspberry Pi/Jetson Nano) to autonomously fly for the Indian Robotics Challenge (IRoC). Tasks include waypoint following, object detection, and autonomous control without GPS.
Team (est.): 8–10 members (Mechatronics & ECE heavy)
Tech Stack: ESP32, ArduPilot/PX4 stack, IMU/GPS/optical flow sensors, Python/C++ for algorithms, possibly ROS.

Line Follower Bot(2022–2023)

A robot designed to follow a predefined track using sensor feedback and embedded control.

Year: ~2022
Description: Tracks and follows a marked line on a surface via optical sensors — fundamental robotics challenge.
Team (est.): 3–4 members
Tech & Tools: IR reflectance sensors, PIC/Arduino microcontroller, basic PID control, robot frame with wheels.

ANAV — Autonomous Navigation Aerial Vehicle (ISRO “Fly Me on Mars”)2025

A fully autonomous aerial vehicle built under the ISRO Robotics Challenge 2025 (“Fly Me on Mars”) with advanced navigation systems such as Extended Kalman Filter, IMU, and real-time decision-making without GPS.

Year: 2025
Description: A custom aerial vehicle tackling NASA/ISRO-style autonomy challenges — includes Kalman filters, IMU fusion, and onboard decision logic.
Team (est.): 8–10 members
Tools/Tech: Extended Kalman Filter (EKF), ROS/DroneKit, IMUs, GPS, Jetson Nano/Pi compute module.

Python • OpenCV • ROS • PX4
Team: A. Kumar, S. Verma
